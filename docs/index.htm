Post Office's within the UK

Analysing UK Post Office Data in OpenStreetMap

The above tool takes Post Office data stored by OpenStreetMap (OSM) and compares it to an officially released data set from the Post Office, allowing for the ability to see if data stored by OSM is accurate.

The data made available by the post office includes post office names, addresses and opening hours.
